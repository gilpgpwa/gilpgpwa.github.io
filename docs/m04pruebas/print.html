1. Instrucciones

  • Las pruebas de unidad permiten asegurar que una parte del código cumple correctamente con sus funciones.

  • Este ejercicio usa la librería PHPUnit para PHP. Puedes profundizar en este tema en la URL https://phpunit.de/

  1. Necesitas tener PHP y composer para PHP instalados y configurados en la ruta de la computadora.

  2. Crea una carpeta y añade los archivos de este ejemplo.

  3. Para el contenido de la carpeta vendor y los archivos composer tienes 2 opciones:

    • Descargar el siguiente archivo zip: composer.zip.

    • Usar un shell como CMD, sh o PowerShell en la carpeta y usar la orden:

      composer require --dev phpunit/phpunit

    • Para ejecutar ls pruebas usa la orden:

      ./vendor/bin/phpunit tests

3. src/recomienda.php

<?php
function
recomienda(string $genero): string
{
 if ($genero === "pop") {
  return "Ariana Grande.";
 } elseif ($genero === "reg") {
  return "Bad Bunny.";
 } else {
  return "De eso no conozco";
 }
}

4. tests/RecomiendaTest.php

<?php

use PHPUnit\Framework\TestCase;

require_once __DIR__ .
 "/../src/recomienda.php";
final class RecomiendaTest
extends TestCase
{
 public function testPop()
 {
  $this->assertEquals(
   "Ariana Grande.",
   recomienda("pop"),
   "Probando pop"
  );
 }
 public function testReg()
 {
  $this->assertEquals(
   "Bad Bunny.",
   recomienda("reg"),
   "Probando reg"
  );
 }
 public function testReg2()
 {
  $this->assertEquals(
   "Daddy Yankee.",
   recomienda("reg"),
   "Probando reg 2"
  );
 }
}

1. Instrucciones

  • Las pruebas de unidad permiten asegurar que una parte del código cumple correctamente con sus funciones.

  • Este ejercicio usa la librería Jest para JavaScript. Puedes profundizar en este tema en la URL https://jestjs.io/

  1. Crear un proyecto de tipo Node.js en https://replit.com.

  2. Desde la pestaña Shell inicializa el proyecto con la orden:

    npm init -y

  3. Instala Jest en el proyecto con la orden:

    npm install --save-dev jest

  4. Añade los archivos de este ejemplo.

  5. Ejecuta el proyecto con la orden:

    jest

  • recomienda.test.js
